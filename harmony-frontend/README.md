# Harmony Frontend

This folder contains the frontend application of the Harmony system.

## Main Responsibilities
- Provides the user interface for participants.
- Displays recommended matches received from the backend.
- Supports multi-language user interaction.

## Structure Overview
- **src/** – Vue components, views, routing, and application logic.
- **public/** – Static public assets.
- **index.html** – Main HTML entry point of the frontend.
