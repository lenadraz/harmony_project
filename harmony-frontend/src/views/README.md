## Views

This directory contains the main user-facing views of the Harmony frontend application.

- **LoginView.vue** – Handles user authentication, allowing users to log in using an phone number and select their preferred language.
- **MatchesView.vue** – Displays personalized match recommendations generated by the backend, including match scores and explanatory insights.
- **SavedView.vue** – Shows matches that the user has saved for future reference or follow-up.
- **MetView.vue** – Lists matches that the user has already met, helping track past interactions.
- **ProfileView.vue** – Allows users to view and manage their personal profile information and preferences.
