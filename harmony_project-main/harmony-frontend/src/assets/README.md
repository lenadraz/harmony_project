## Assets

This directory contains static assets and global styling files used across the Harmony frontend.

- **base.css** – Defines global base styles, CSS resets, and default typography used throughout the application.
- **main.css** – Contains the main global styling rules, including layout, color themes, and shared UI components.
- **default-avatar.png** – Default profile image displayed when a user has not uploaded a personal avatar.
- **harmony-logo.png** – Official Harmony project logo used in the application interface.
- **logo.svg** – Scalable vector version of the application logo, optimized for different screen sizes and resolutions.
