# Backend Service

This folder contains the backend server of the Harmony system, responsible for handling requests, managing data, and coordinating the matching logic.

## Folder and Files Overview
- **index.js** – Main server entry point that initializes the backend application.
- **similarity.js** – Implements similarity calculations used for participant matching.
- **llmExplanation.js** – Generates textual explanations for match recommendations using language models.
- **data/** – Contains datasets and input files used by the backend.
- **.env** – Environment variables configuration file.
- **package.json** – Backend dependencies and scripts configuration.
